# Prompts de curadoria

Esta página reúne os templates de prompt utilizados nas tarefas de curadoria
automatizada do projeto. Esses templates definem a estrutura das instruções
enviadas ao modelo durante etapas como classificação de dificuldade e
identificação da legislação base.

Para visualizar os prompts utilizados no projeto, acesse o link abaixo:

- [Clique neste link para visualizar os prompts](/resultados/prompts)

## Engine de templates

Os prompts são renderizados com **MiniJinja**, o que permite inserir
dinamicamente os dados de cada questão no template antes da execução.

Essa abordagem facilita a padronização das entradas enviadas ao modelo e ajuda a
manter o processo de curadoria mais consistente, reutilizável e reprodutível.
