# Referências

Esta página reúne os principais links utilizados no projeto, incluindo datasets,
modelos, bibliotecas, artigos e ferramentas de apoio.

## Datasets

- [maritaca-ai/oab-bench](https://huggingface.co/datasets/maritaca-ai/oab-bench):
  dataset OAB Bench no Hugging Face
- [eduagarcia/oab_exams](https://huggingface.co/datasets/eduagarcia/oab_exams):
  dataset OAB Exams no Hugging Face

## Modelos

Os modelos abaixo foram utilizados nas execuções locais realizadas no projeto:

- [llama3.2:3b](https://ollama.com/library/llama3.2%3A3b):
  modelo disponibilizado no Ollama
- [gemma2:2b](https://ollama.com/library/gemma2%3A2b):
  modelo disponibilizado no Ollama
- [qwen2.5:3b](https://ollama.com/library/qwen2.5%3A3b):
  modelo disponibilizado no Ollama

## Bibliotecas

- [Ollama](https://ollama.com/):
  ferramenta para execução local de modelos de linguagem
- [MiniJinja](https://github.com/mitsuhiko/minijinja):
  engine de templates utilizada na renderização dos prompts
- [Evaluate](https://huggingface.co/docs/evaluate/index):
  biblioteca de métricas do ecossistema Hugging Face
- [BERTScore](https://github.com/Tiiiger/bert_score):
  implementação da métrica BERTScore para avaliação semântica
- [Typer](https://typer.tiangolo.com/):
  biblioteca para construção de CLIs em Python
- [UV](https://docs.astral.sh/uv/):
  gerenciador de dependências e ambientes Python
